# JobShield AI

AI-powered fake job detection platform using Machine Learning and NLP.

## Features
- Fake job detection
- Netflix-style UI
- Login & Signup system
- AI confidence gauge
- Dashboard analytics
- Cybersecurity-inspired design

## Technologies Used
- Python
- Streamlit
- Machine Learning
- NLP
- Scikit-learn
- Plotly
- SQLite

## Run Project

pip install -r requirements.txt

streamlit run app.py
